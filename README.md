# goApp
该工程主要为个人一些golang小工具。目前包括以下三个子工程：

	windowFileCleaner
	lianjiaCrawler
	huaweiWebSnap
  
windowFileCleaner 
--------------------------------------------------------------------------------------------------

	该工具由于一般的垃圾清理工具都无法对c盘做一个文件清理，所以通过该小应用查看相应的文件夹大小，手动对一些大文件进行清理操作。
